3D Game of Life

Generation: 0



Conway's Game of Life is a game with simple rules: (1) Any living cell with 2 or 3 neighbors lives on. (2) Any dead cell with exactly 3 live neighbors becomes a live cell.

I wanted to build an infinite, "three dimensional" game by turning the finite 2D grid into a cube. A fun edge case: some patterns that live on forever in an infinite 2D grid might eventually end here. This is because most cells have 8 neighbors, but in this version the corner cells only have 7 neighbors. Because math.
